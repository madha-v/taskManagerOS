# taskManagerOS
A replica of windows task manager system.
# Mini Task Manager - Pro

## Quick Start:
1. Install Python 3.8+ from python.org
2. Unzip project folder
3. Open Command Prompt in folder
4. Install dependencies: pip install -r requirements.txt
5. Run: python main.py

## Build Executable:
1. Run build-win.bat
2. Executable will be in dist\mini_task_manager\mini_task_manager.exe
